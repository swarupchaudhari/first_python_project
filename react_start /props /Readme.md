React Profile Card Application

📌 Overview

A simple and responsive React application that demonstrates the use of reusable components and props. The project displays multiple user profile cards with profile images, names, ages, descriptions, and action buttons.

This project is ideal for beginners learning React fundamentals such as Components, Props, JSX, and CSS Styling.

---

🚀 Features

- Reusable React Components
- Dynamic Data Using Props
- Responsive Card Layout
- Modern UI Design
- Profile Image Display
- User Information Rendering
- Clean and Maintainable Code Structure

---

🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- JSX
- CSS3
- Vite / Create React App
