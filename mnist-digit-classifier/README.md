# MNIST Digit Classifier

This project builds a simple Neural Network using TensorFlow to classify handwritten digits from the MNIST dataset.

## Features
- Uses TensorFlow and Keras
- Trains on 60,000 images
- Tests on 10,000 images
- Achieves ~98% accuracy

## Dataset
MNIST handwritten digits dataset (0-9)

Image size: 28x28 pixels

## Model Architecture

Input Layer: 28x28  
Flatten Layer  
Dense Layer: 128 neurons (ReLU)  
Output Layer: 10 neurons (Softmax)
